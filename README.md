# redcat-manage

<br>

> 1、基于vue-cli + Element构建的前端权限管理页 <br>

> node版本6.x+ , npm版本3.x+

> 注：webpack使用4.x+版本，构建方式跟3.x有很大区别，建议查看官方文档
<br>

# 开始使用



```
安装依赖
npm install
```
<br>

```
运行
npm run dev
```
<br>

```
浏览器打开 http://localhost:9527.
```

<br>


# 打包&发布

```
构建到测试环境
npm run build:sit
```
<br>

```
构建到生产环境
npm run build:prod
```
<br>
